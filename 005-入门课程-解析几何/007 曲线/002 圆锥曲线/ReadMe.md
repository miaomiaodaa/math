**说明2**
线-面夹角: $[0,\dfrac{\pi}{2}]$

截平面与旋转轴夹角: $\alpha=\dfrac{\pi}{2}\Rightarrow$截线是圆

截平面与旋转轴夹角: $\theta<\alpha<\dfrac{\pi}{2}\Rightarrow$截线是椭圆

截平面与旋转轴夹角: $\theta=\alpha\Rightarrow$截线是抛物线

截平面与旋转轴夹角: $\theta>\alpha\Rightarrow$截线是双曲线

特别的，当截面过圆锥定点时

截线图形为: 点、一条直线、两条直线
